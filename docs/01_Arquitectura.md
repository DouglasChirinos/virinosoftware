# 01 - Arquitectura

## Capas

1. app: interfaz grafica y controladores.
2. engine.geometry: primitivas geometricas.
3. engine.measurements: validacion y modelos de medidas.
4. engine.drafting: contratos y generadores de patron.
5. engine.garments: reglas especificas por prenda.
6. exports: salidas SVG, DXF y PDF.

## Regla critica

La GUI nunca debe contener formulas de patronaje.
