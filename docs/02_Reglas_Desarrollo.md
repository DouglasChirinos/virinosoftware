# 02 - Reglas de Desarrollo

## Reglas obligatorias

- Usar Python 3.
- Tipar funciones publicas.
- Usar dataclasses para primitivas simples.
- Usar Pydantic para validar datos de entrada.
- Mantener pruebas en tests/.
- No mezclar GUI con motor geometrico.
- Todo calculo de patronaje debe ser testeable sin abrir la interfaz.
